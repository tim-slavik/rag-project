# Mini RAG

A minimal Retrieval‑Augmented Generation pipeline demonstrating document scraping, text extraction, embedding, and retrieval.  I added the scraping portion as a way to personalize this a little and do something I haven't done before.

## Features
- PDF/DOCX scraping
- Text extraction + cleaning
- FAISS vector search
- Simple RAG query pipeline
